# Results

This folder contains all numerical outputs from the seven experiment notebooks, together
with the pilot data used to calibrate anchor values for the narrative document experiments.

## Contents

Each experiment family produces one or more CSV files containing the parsed numeric
responses from all five models across all conditions and documents. File naming follows
the convention `<source_paper>_<study>.csv` where applicable.

- `pilot_anchors.json` — baseline effort estimates for each of the 34 full-length narrative
  requirements documents, produced by a separate pilot model cohort under a control
  condition. These values were used to calibrate low and high anchor values for the
  Aranda and Løhre experiment families. See Section 3.3.1 of the dissertation for details.

## Regenerating Results

Results can be regenerated by running the notebooks in the `experiments/` folder. The
`requirements_documents/` folder and `utils.py` must be present in the same directory as
the notebooks. Results will be written back to this folder.
