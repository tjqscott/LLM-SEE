# Figures

This folder contains all figures used in the dissertation, together with the script used
to generate them.

## Contents

- `visualise_results.py` — generates all figures from the experimental results. Must be
  run from the same directory as the `results/` folder.
- 20 PNG files, one per figure appearing in the dissertation.

## Regenerating Figures

Figures can be regenerated by running `visualise_results.py` with the `results/` folder
present in the same directory:

```bash
python visualise_results.py
```

Note that regenerated figures may differ slightly from those provided here if results have
been regenerated, for reasons discussed in Section 3.2 of the dissertation.
