# Statistical Analysis

This folder contains the R script used to perform all statistical analyses reported in
the dissertation, together with its output.

## Contents

- `analysis.R` — performs all statistical tests reported in the dissertation, including
  paired Wilcoxon signed-rank tests, Rom's correction for family-wise error rate, and
  Cliff's δ effect sizes. Must be run from the same directory as the `results/` folder.
- `output.md` — the output produced by `analysis.R`, containing all test statistics,
  corrected p-values, and effect sizes reported in the dissertation.

## Requirements

R can be obtained from https://www.r-project.org/. The following packages are required
and can be installed from the R console:

```r
install.packages("jsonlite")
install.packages("effsize")
```

## Regenerating Output

The output can be regenerated by running `analysis.R` with the `results/` folder present
in the same directory.
